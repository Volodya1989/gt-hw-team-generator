## Installation

1. Clone the repo.
2. Run npm install command in your terminal/bash.
3.  Then run  node app.js command in your terminal/bash.

## Usage

This Team Generator was created as command line application for those people who want quickly generate a webpage that displays their team's basic info, so that they can quickly access emails and GitHub profiles.
